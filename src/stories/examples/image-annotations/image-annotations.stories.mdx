# Image annotations

A small **design-review** style app on top of the library: pan and zoom the
photo, then drop **numbered pins** or **on-canvas labels** at the correct spot
even while zoomed in.

Clicks are converted from **viewport coordinates** into **content space** using
the live `positionX`, `positionY`, and `scale` from the transform state
(`translate` then `scale`, origin top-left), so markers stay glued to the image.

Use **Navigate** vs **Pin / Label** to toggle panning — while placing notes,
panning is disabled so the next click creates a draft. **Focus** in the thread
calls **`zoomToElement`** on the marker’s wrapper.
