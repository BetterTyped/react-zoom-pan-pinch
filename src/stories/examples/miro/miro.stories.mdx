# Miro

A **Miro-style** canvas interaction: hold `Cmd` (or `Ctrl`) and scroll to
zoom, or use the trackpad freely to pan. The `activationKeys` option accepts
a function that receives the currently held keys, letting you create
conditional gesture activation.

The board below is built with HTML/CSS (sticky notes, frames, flow steps, and a
light dot grid) on a **large infinite-style canvas** so zoom and pan feel like a
real whiteboard tool.

This pattern separates zoom from scroll — ideal for infinite canvas apps
where the default scroll behavior should pan, not zoom.
