# useTransformEffect

Run a **side-effect** on every transform change — without triggering React
re-renders. Think of it as `useEffect` for transform state: your callback
fires on every pan, zoom, or pinch event, and can optionally return a cleanup
function.

## Signature

```ts
function useTransformEffect(
  callback: (state: ReactZoomPanPinchContextState) => void | (() => void),
): void
```

**`state`** contains:

```ts
{
  state: { previousScale, scale, positionX, positionY };
  instance: ReactZoomPanPinchContext;
}
```

The callback can return a **cleanup function** that runs before the next
invocation and on unmount — the same pattern as `useEffect`.

## When to Use

- **Analytics** — log zoom events or viewport positions
- **URL sync** — update query parameters with the current viewport
- **External state** — push transform data to a store (Redux, Zustand, etc.)
- **DOM mutations** — update a canvas overlay or external tooltip positions
- **Debounced saves** — persist viewport state after interaction stops

For rendering UI that updates with transform state, use
`useTransformComponent` instead — it manages React state and re-renders for you.

## Examples

### Sync to URL

```tsx
import { useTransformEffect } from "react-zoom-pan-pinch";

function ViewportSync() {
  useTransformEffect(({ state }) => {
    const params = new URLSearchParams(window.location.search);
    params.set("x", String(Math.round(state.positionX)));
    params.set("y", String(Math.round(state.positionY)));
    params.set("s", state.scale.toFixed(2));
    window.history.replaceState(null, "", `?${params}`);
  });

  return null;
}
```

### Analytics Logging

```tsx
function ZoomAnalytics() {
  useTransformEffect(({ state }) => {
    if (state.scale !== state.previousScale) {
      analytics.track("zoom_changed", { scale: state.scale });
    }
  });

  return null;
}
```

### Cleanup Pattern

```tsx
function Overlay() {
  useTransformEffect(({ state }) => {
    const overlay = document.getElementById("canvas-overlay");
    overlay?.style.setProperty("--scale", String(state.scale));

    return () => {
      overlay?.style.removeProperty("--scale");
    };
  });

  return null;
}
```

## Key Differences

| | `useTransformEffect` | `useTransformComponent` |
|---|---|---|
| Purpose | Side-effects | Render UI |
| Returns | `void` | `T` (your callback's return) |
| Re-renders? | No | Yes |
| Use case | Logging, syncing, DOM | Badges, labels, indicators |

## Requirements

Must be called inside a descendant of `TransformWrapper`.
