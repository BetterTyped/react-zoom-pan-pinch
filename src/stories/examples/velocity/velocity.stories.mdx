# Pan velocity (inertia)

After a fast pan the library applies **momentum** so the canvas keeps
moving and **eases out** — the same physics you feel when scrolling on
mobile. Toggle **Momentum / Locked**, adjust **Sensitivity** and
**Friction**, then fling the starfield to feel the difference.

| Prop | Effect |
|---|---|
| `velocityAnimation.disabled` | Turn inertia on/off |
| `velocityAnimation.sensitivityMouse` | How much initial speed is captured (higher = faster fling) |
| `velocityAnimation.inertia` | Decay coefficient — higher = longer glide |
| `velocityAnimation.animationTime` | Base animation duration in ms |
