# Center Zoomed Out

When the image is **zoomed out** (scale below 1×), the content automatically
**stays centered** in the viewport. Try panning — the image snaps back. Once
you zoom in past 1×, free panning works normally.

This is the same pattern used in cinema / ticket-booking UIs where the layout
should always stay framed when the user hasn't zoomed in yet.
