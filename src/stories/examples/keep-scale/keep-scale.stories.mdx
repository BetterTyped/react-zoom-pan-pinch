# Keep Scale

The `KeepScale` component wraps child elements so they **maintain their visual
size** regardless of the current zoom level. As you zoom into the map, the
markers stay the same size on screen while the underlying content scales
normally.

This is essential for map pins, annotations, tooltips, or any overlay element
that should remain readable at all zoom levels. Try zooming to 100x — the
markers stay crisp and legible while the map image scales up.
