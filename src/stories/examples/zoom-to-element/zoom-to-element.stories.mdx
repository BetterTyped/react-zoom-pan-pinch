# Zoom To Element

Programmatically animate the viewport to **focus on a specific element** inside
the transformed content. Click any card on the canvas or use the target buttons
in the control bar — `zoomToElement(elementId)` smoothly pans and zooms to frame
that element in view.

This is the core API for seat-map selectors, diagram node focus, annotation
highlights, and any "navigate to target" interaction pattern.
