# Virtualize

Mount and unmount children **based on viewport visibility**. `Virtualize` wraps
content positioned inside the transformed canvas and only renders its children
when the declared bounding box overlaps the viewport. This is ideal for large
canvases with many elements — seat maps, tile grids, node graphs — where keeping
every item in the DOM would hurt performance.

## How it works

Each `Virtualize` component declares its content-space position and size via
`x`, `y`, `width`, and `height` props. On every transform change (pan, zoom,
pinch) the component checks whether the scaled and translated bounding box
intersects the viewport. When it does, children are mounted; when it doesn't,
children are removed from the DOM.

## Quick start

```tsx
<TransformWrapper>
  <TransformComponent>
    <div style={{ position: "relative", width: 4000, height: 4000 }}>
      <Virtualize x={100} y={200} width={300} height={150} margin={50}>
        <MyExpensiveWidget />
      </Virtualize>
    </div>
  </TransformComponent>
</TransformWrapper>
```
