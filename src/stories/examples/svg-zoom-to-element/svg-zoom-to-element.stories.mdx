# SVG Zoom To Element

Target elements **inside an SVG** by their `id` attribute. The control bar
buttons zoom to three different paths within the illustration. This works the
same way as HTML element targeting — `zoomToElement` finds the element by its
`id` and calculates the bounding box to frame it in view.
