# useControls

Simple and easy access to the helpers inside of the `TransformWrapper`.

### Example

```tsx
const App = () => {
  // Simple access to the helpers
  const { instance, zoomIn, zoomOut, ...rest } = useControls();

  // ...

  return (
    <button type="button" onClick={() => zoomIn()}>
      Zoom in
    </button>
  );
};
```
